# PC 后台 UI 规范

版本：v2.0 (Arco Design 完整对标版)  
适用范围：校园智能积分兑换终端管理后台 PC 端页面，覆盖数据中心、配置管理、列表页、复杂表单、配置画布、详情页及所有通用组件。  
设计基准：本规范全面对标字节跳动企业级设计系统 **Arco Design**，旨在通过极致清晰的 Token 定义、二十余个核心组件的精细尺寸、边距、对齐与状态约束，为 Agent 提供无死角的设计与编码标准，消除界面设计中的随意性。

---

## 1. 设计原则

### 1.1 专业克制
- 界面设计以信息读取与操作效率为唯一核心。不采用渐变色、大投影、粗边框或非功能性装饰性元素。
- 圆角与间距保持高度统一，以网格化布局承载数据，使界面风格利落、扁平且富含呼吸感。

### 1.2 信息优先与对齐一致
- 对齐是后台设计的基石。所有卡片边界、表单项左边缘、表格列边沿必须在垂直或水平方向精确对齐。
- **列对齐一致性**：表格的表头与表体单元格对齐方式必须绝对一致（禁止表头居中而内容靠左）。

### 1.3 间距网格化
- 严禁使用 10px、15px、20px 等即兴数值。页面所有的 margin（外边距）与 padding（内边距）必须符合 **8px 步长（结合 4px 微调）**的间距系统。

## 2. 设计 Token (Arco Design 标准)

### 2.1 字体 (Typography)

| Token | 值 | 用途 |
|---|---|---|
| `font-family-base` | `Inter`, `-apple-system`, `BlinkMacSystemFont`, `PingFang SC`, `Microsoft YaHei`, sans-serif | 系统默认字体集 |
| `font-weight-regular` | 400 | 面包屑、正文、表单值、输入框文本、表格体文本 |
| `font-weight-medium` | 500 | 页面标题、卡片次标题、数值、重点字段 |
| `font-weight-semibold` | 600 | 区块主标题、表格表头文本、Modal 标题 |

### 2.2 字号与行高 (Font Size & Line Height)

| Token级 | 像素 (Size) | 字重 (Weight) | 行高 (Height) | 推荐中性色 | 适用场景 |
|---|---|---|---|---|---|
| `font-size-small` | 12px | 400 | 20px | `#86909C` / `#F53F3F` | 错误校验提示、输入框计数、表单辅助说明 |
| `font-size-body-2` | 13px | 400 | 20px | `#4E5969` | 次要正文、面包屑历史项、轻提示辅助文字 |
| `font-size-body-1` | 14px | 400 | 22px | `#1D2129` | 系统标准正文、输入框值、表单 Label、表格单元格 |
| `font-size-title-1` | 14px | 600 | 22px | `#1D2129` | 表格表头、简易分组卡片次标题 |
| `font-size-title-2` | 16px | 600 | 24px | `#1D2129` | 卡片标题、区块主标题、Modal 标题 |
| `font-size-title-3` | 18px | 500 | 26px | `#1D2129` | 页面主标题（面包屑下方） |

### 2.3 色彩系统 (Color)

本系统颜色**全量对标 Arco Design**：

#### 2.3.1 品牌色与状态色
- **主色 (Primary-6)**：`#165DFF`（主要按钮、激活态文字、焦点描边）
- **主色悬浮 (Primary-5)**：`#4080FF`
- **主色按下 (Primary-7)**：`#0E42D2`
- **主色浅激活 (Primary-1)**：`#E8F3FF`（选中下拉项背景、选中表格行背景）
- **成功色 (Success-6)**：`#00B42A`（浅背景：`#E8FFEA`，用于正常/成功状态）
- **警告色 (Warning-6)**：`#FF7D00`（浅背景：`#FFF7E6`，用于待处理/警示状态）
- **错误/危险 (Danger-6)**：`#F53F3F`（浅背景：`#FFECE8`，用于报错、必填星号、删除操作）

#### 2.3.2 中性色阶 (Arco Neutral Color)
- **主要文本 (`color-text-title`)**：`#1D2129`（标题、高亮文字、常规表单 Label，灰 10）
- **正文文本 (`color-text-body`)**：`#4E5969`（常规段落、表单输入值、次要正文，灰 8）
- **次要/辅助 (`color-text-secondary`)**：`#86909C`（辅助提示、未选面包屑、输入框占位符，灰 6）
- **禁用文本 (`color-text-disabled`)**：`#C9CDD4`（禁用控件文字，灰 4）
- **常规边框 (`color-border-default`)**：`#E5E6EB`（输入框/下拉框/普通按钮的默认边框，灰 3）
- **分割线 (`color-divider`)**：`#F2F3F5`（普通分割线、表格网格横线、列表项分割线，灰 2）
- **浅填充/背景 (`color-bg-fill`)**：`#F7F8FA`（表头背景、禁用输入框背景，灰 1）
- **页面背景 (`color-bg-page`)**：`#F2F3F5`（最底层大背景）
- **容器背景 (`color-bg-container`)**：`#FFFFFF`（卡片底色、Modal 内部白底）

### 2.4 圆角系统 (Radius)

- **小控件圆角**（输入框、选择器、小型按钮、Tag 标签）：统一为 **4px**。
- **大容器圆角**（页面卡片 Card、对话框 Modal、抽屉 Drawer）：统一为 **8px**。
- **无圆角**（全宽通栏组件，如顶部 Header 导航、通栏分栏线）：**0px**。
- **胶囊标签圆角**（如状态 Badge、特定分类 Tag）：**16px** 或半圆角。

### 2.5 间距系统 (Spacing)

| Spacing Token | 像素值 | 适用布局场景 |
|---|---|---|
| `spacing-mini` | 4px | 极紧邻关联（如必填星号与文字、小 Icon 与旁边文本） |
| `spacing-small` | 8px | 紧邻同级元素（如并排按钮间距、Tag 标签间距、单选/复选框组内间距） |
| `spacing-close` | 12px | 偏紧密垂直关联（如面包屑与下方主标题、过滤栏内 Label 与控件垂直距离） |
| `spacing-medium` | 16px | 模块内标准布局（表单行垂直间距、卡片内小组件间距、小卡片间距） |
| `spacing-large` | 24px | 容器内留白（页面视口四周留白 padding、卡片内部 padding、Modal 主体 padding） |
| `spacing-xlarge` | 32px | 大区块自然分割间距 |

## 3. 页面大框架与布局规范 (Layout)

### 3.1 侧边栏与头部
- **侧边导航栏 (Sider)**：宽度固定为 **200px**。折叠后宽度固定为 **48px**。
- **顶部工作栏 (Header)**：高度固定为 **48px**，下边缘带 1px 分割线 `#F2F3F5`。
- **工作区 (Content)**：页面大背景色为 `#F2F3F5`。内容卡片与视口边缘 padding 统一为 **24px**。

### 3.2 卡片容器 (Card)
- 背景白底 `#FFFFFF`，圆角 **8px**。
- 卡片内部 padding 统一为 **20px**。
- 卡片与卡片之间的垂直间距统一为 **16px**。
- 卡片默认不使用明显的深投影，阴影规范为：`box-shadow: 0 4px 10px rgba(0, 0, 0, 0.04)`。

### 3.3 页面标题与卡片布局约束 (Arco Pro 最佳实践)
- **页面标题置内与悬空防错规则**：
  - **列表页**：页面标题（如“成绩管理”、“查询表格”）**严禁**作为悬浮于灰色底盘上的独立块。它们必须作为白色大卡片容器内部的第一行标题（`16px`，`font-weight-semibold`，颜色 `#1D2129`），与下方的查询控件保持紧凑间距。
  - **编辑/详情页**：页头仅保留小巧精致的面包屑（如 `数据中心 / 成绩管理 / 新建考试`）漂浮在灰色底盘上。卡片外部**不应**存在大号的页面标题（如 `新建考试` H2 标题），应由卡片内部第一个带品牌蓝色标识的区块标题（如“基本信息”）来主导页面的第一视觉。
- **列表页单卡片归一规范**：
  - 查询列表页（包含查询筛选和数据表格的页面）**严禁**拆分为“筛选卡片”和“表格卡片”双容器。
  - 必须归并为**同一个大白色卡片容器**，将主标题、查询表单行、操作栏和表格全部包裹在一起，中间通过 1px 细线 `border-t border-[#F2F3F5]` 进行区隔，确保结构高度集成和视觉紧凑感。

## 4. 20+ 核心组件规范细则

本节定义了中后台所有高频通用组件的尺寸、边距、字体与颜色标准，Agent 在编写 DOM 样式时必须严格遵守。

### 4.1 基础与布局组件

#### 4.1.1 Grid 栅格
- 统一使用 **24 栅格**系统。
- 筛选过滤栏 Gutter (横向列间距) 固定为 **16px**。
- 复杂表单两列布局 Gutter 固定为 **24px**。

#### 4.1.2 Space 间距组件
- 水平或垂直并排元素时，间距设置为三档：
  - `small`（紧凑）：**8px**
  - `medium`（默认）：**16px**
  - `large`（宽松）：**24px**

#### 4.1.3 Divider 分割线
- 水平分割线：粗细 **1px**，颜色 `#F2F3F5`，上下 margin 统一为 **16px**。
- 垂直分割线（如操作按钮间分割）：粗细 **1px**，高度 **12px**，颜色 `#E5E6EB`，左右 margin 统一为 **8px**。

---

### 4.2 导航组件

#### 4.2.1 Menu 菜单
- **侧边菜单项高度**：统一为 **40px**，左右 margin 为 **8px**（即左右留白，内部 padding 宽度自适应）。菜单项圆角为 **4px**。
- **交互状态色**：
  - 默认：文字 `#4E5969`，背景透明。
  - Hover 悬浮：文字 `#1D2129`，背景 `#F2F3F5`。
  - Active 选中：文字主色 `#165DFF`，背景主色浅激活色 `#E8F3FF`，且右侧或左侧带有一条宽 3px 的主色竖线。
  - 禁用：文字 `#C9CDD4`，背景透明。

#### 4.2.2 Breadcrumb 面包屑
- 面包屑高度由内容自适应，下边缘与页面标题的垂直间距固定为 **12px**。
- 历史路径项文字颜色为 `#86909C`，悬浮变 `#1D2129`；当前页面项文字颜色为 `#1D2129`。
- 分隔符统一使用 `/`，左右外边距为 **8px**，分隔符颜色为 `#C9CDD4`。

#### 4.2.3 Pagination 分页
- 页码按钮高度固定为 **32px**，最小宽度为 **32px**。
- 页码与页码按钮之间的横向间距固定为 **8px**，按钮圆角为 **4px**。
- **状态色**：
  - 默认按钮：白底，边框 `#E5E6EB`，文字 `#4E5969`。
  - 选中页码：背景主色 `#165DFF`，无边框，文字 `#FFFFFF`。
  - Hover 悬浮：边框变 `#165DFF`，文字变 `#165DFF`。
  - 禁用状态：背景 `#F7F8FA`，文字 `#C9CDD4`，边框 `#E5E6EB`。

#### 4.2.4 Steps 步骤条
- 步骤条每个节点间距自适应，步骤小图标（圆形）直径为 **28px**。
- 步骤名称字号 `14px`，未激活步骤文字 `#86909C`，激活步骤 `#1D2129`（加粗 600），完成步骤 `#165DFF`。
- 步骤间的连线高度为 **1px**，颜色在激活状态前为 `#165DFF`，未进行步骤为 `#E5E6EB`。

### 4.3 数据输入组件

#### 4.3.1 Button 按钮
- 按钮圆角一律为 **4px**。
- **尺寸与内边距**：
  - **大 (Large)**：高度 **36px**，左右 padding **16px**，字号 **14px**。
  - **中 (Medium/默认)**：高度 **32px**，左右 padding **15px**，字号 **14px**。
  - **小 (Small)**：高度 **28px**，左右 padding **12px**，字号 **12px**。
- **类型与状态色**：
  - **主按钮 (Primary)**：背景 `#165DFF`，文字 `#FFFFFF`。悬浮 `#4080FF`，按下 `#0E42D2`，禁用（背景 `#F7F8FA`，文字 `#C9CDD4`）。
  - **次要按钮 (Secondary)**：背景 `#FFFFFF`，边框 1px `#E5E6EB`，文字 `#4E5969`。悬浮（边框与文字变 `#165DFF`）。
  - **文字按钮 (Link/Text)**：无边框无背景。文字 `#165DFF`，悬浮变 `#4080FF`。

#### 4.3.2 Input & InputNumber 输入框与数字输入框
- 高度统一为 **32px**，圆角 **4px**。
- 左右内边距固定为 **12px** (`padding: 0 12px`)。
- **状态样式**：
  - 默认：白底 `#FFFFFF`，边框 1px `#E5E6EB`，文字 `#1D2129`，占位文本 `#C9CDD4`。
  - 聚焦 (Focus)：边框 `#165DFF`，且外发光阴影 `box-shadow: 0 0 0 2px rgba(22, 93, 255, 0.2)`。
  - 禁用 (Disabled)：背景 `#F7F8FA`，文字 `#C9CDD4`，边框 `#E5E6EB`。
  - 错误校验状态：边框变 `#F53F3F`，外发光变 `rgba(245, 63, 63, 0.2)`。

#### 4.3.3 Select 下拉选择器
- 尺寸（高度 32px，圆角 4px，边框 `#E5E6EB`）与 Input 保持完全一致。
- 右侧下拉三角形 Icon 尺寸为 **14px**，颜色 `#86909C`。
- 下拉弹层：
  - 背景色 `#FFFFFF`，圆角 **4px**，阴影为 `box-shadow: 0 4px 10px rgba(0, 0, 0, 0.08)`，最大高度 **260px**（超出自动滚动）。
  - 选项行 (Option) 高度固定为 **32px**，左右内边距 **12px**。Hover 背景色为 `#F2F3F5`，选中态文字颜色为 `#165DFF`，背景色为 `#E8F3FF`。

#### 4.3.4 Cascader 级联选择器
- 级联输入框尺寸对标 Select，高度 32px，圆角 4px。
- 下拉展开的多列面板，每列宽度固定为 **160px**，各面板间带 1px 垂直分割线 `#F2F3F5`，选中项文字为主色 `#165DFF`。

#### 4.3.5 Switch 开关
- 开关整体为圆角胶囊状，默认尺寸：高度 **22px**，宽度 **40px**。
- 内部白色圆形滑块直径为 **18px**，滑块在左侧/右侧的切换过渡动效时间为 `0.2s`。
- **状态色**：
  - 开启状态：背景色为主色 `#165DFF`。
  - 关闭状态：背景色为 `#C9CDD4`。
  - 禁用状态：开启状态变 `#E8F3FF`，关闭状态变 `#F2F3F5`。

#### 4.3.6 Radio & Checkbox 单选与复选框
- 选框圆形或方形边框尺寸固定为 **16px × 16px**。
- 选框与后面选项文字的横向间距固定为 **8px**，选项与选项之间的横向间距固定为 **16px**。
- **交互状态色**：
  - 选中：边框及内部填充变为 `#165DFF`，文字颜色不变 `#1D2129`。
  - 未选中：边框 1px `#E5E6EB`，背景 `#FFFFFF`。
  - 禁用：背景 `#F7F8FA`，边框 `#E5E6EB`，文字变 `#C9CDD4`。

#### 4.3.7 DatePicker 日期与时间选择器
- 单个日期选择框：高度 **32px**，宽度固定为 **220px**，右侧带日历图标（14px，`#86909C`）。
- 范围日期选择框：高度 **32px**，宽度固定为 **320px**。
- 下拉日历面板：宽度固定为 **280px**，面板圆角 **4px**，带标准阴影。当前选中天为 `#165DFF` 圆形背景，今日文字为 `#165DFF`（无背景色，带下方小圆点）。

#### 4.3.8 Upload 上传
- 按钮式上传：高度对标标准 Button (32px)。
- 卡片拖拽式上传区域：尺寸固定为 **120px × 120px**，圆角为 **4px**。边框为 1px 虚线 `#E5E6EB`，背景色为 `#F7F8FA`。拖拽悬浮状态下，虚线边框变更为 `#165DFF`。

### 4.4 数据展示组件

#### 4.4.1 Table 通用表格
- 表格宽度默认 **100%**。单元格网格线粗细 **1px**，颜色 `#F2F3F5`。
- **表格行高与密度**：
  - 常规 (Medium)：表头和表体行高均固定为 **48px**（上下 padding 12px，左右 padding 16px）。
  - 紧凑 (Small)：行高固定为 **40px**（上下 padding 8px，左右 padding 16px）。
- **列对齐一致性与 Arco 默认靠左规范**：
  - **通用靠左对齐原则**：除了特定的金额/纯数字对比列之外，列表表格中的所有字段（包括**常规文本、状态标签列、操作按钮列、日期时间列**）默认**统一采用靠左对齐**（`text-left`），表头与表体对齐方式必须绝对一致。
  - **右对齐例外**：仅在包含高精度数值、统计金额或用于纵向大小比对的纯数字列时，才使用**右对齐**（`text-right`），此时表头也必须对应设为右对齐。
- **宽度保护 min-width 分配**：
  - 序号/复选框列：`60px`。
  - 姓名/科目/分类标签：`120px`。
  - 时间列：`160px` ~ `180px`。
  - 操作列：根据文字链接按钮的个数：1 个 `70px`；2 个 `120px`；3 个 `160px`。超过 3 个按钮应折叠为“更多”下拉弹窗。
- **表格表头样式**：背景色 `#F7F8FA`，字号 `14px`，字重 600，颜色 `#1D2129`。

#### 4.4.2 Tag 标签
- 默认高度固定为 **24px**，左右 padding 为 **8px**，圆角为 **2px**。
- 背景色采用浅色功能色阶（如成功态背景 `#E8FFEA`，文字 `#00B42A`，边框无或 `#AFF0B5`）。
- **状态胶囊标签**：高度固定为 **24px**，圆角固定为 **12px**（半圆形），用于高频状态标识。

#### 4.4.3 Badge 徽标
- 徽标红点直径为 **6px**。
- 数字徽标（如未读消息数）：高度固定为 **18px**，圆角 **9px**，左右最小 padding **6px**。背景色为红 `#F53F3F`，文字 `#FFFFFF`（字号 12px，字重 600）。
- 徽标定位在依附对象（头像、图标）右上角，中心偏出对象右上边缘 `4px`。

#### 4.4.4 Card 卡片
- 白色背景 `#FFFFFF`，圆角 **8px**，卡片内 padding 固定为 **20px**。卡片之间垂直间距为 **16px**。

#### 4.4.5 Tooltip & Popover 气泡提示
- **Tooltip（文字气泡）**：背景为深黑 `#1D2129`（透明度 90%），圆角 **4px**，文字为 12px `#FFFFFF`，内边距 `6px 10px`。气泡小三角高 `6px`。
- **Popover（复杂气泡）**：白底背景 `#FFFFFF`，圆角 **4px**，带标准阴影。内部包含标题（14px，加粗 600，`#1D2129`）与内容段落（13px，`#4E5969`），内边距统一为 `12px 16px`。

#### 4.4.6 Avatar 头像
- 提供三种标准规格：小 **24px** / 中 **32px** (默认) / 大 **40px**。
- 形状可选择**圆形** (border-radius: 50%) 或**方形** (border-radius: 4px)。

### 4.5 反馈与弹窗组件

#### 4.5.1 Alert 警告提示
- 用于卡片、页面顶部的常驻提示。
- 整体上下 padding 固定为 **12px**，左右 padding 固定为 **16px**，圆角 **4px**。
- 包含状态图标（16px）与提示内容，文字字号 14px，颜色依据状态映射（如成功态背景 `#E8FFEA`，文字 `#00B42A`）。

#### 4.5.2 Drawer 抽屉
- 从屏幕右侧拉出，默认宽度：中等页签 **360px**，复杂表单页签 **520px**。
- 抽屉遮罩层颜色为 `#1D2129`（60% 透明度）。
- 头部高 48px，主体区左右 padding 24px，上下 padding 20px。

#### 4.5.3 Modal 对话框
- 遮罩层颜色 `#1D2129`（60% 透明度）。Modal 容器背景 `#FFFFFF`，圆角 **8px**。
- 尺寸宽度推荐：普通信息框 **520px**，复杂数据填报框 **840px**。
- **各分区边距与高度**：
  - **头部 (Header)**：高度自适应，左右 padding **20px**，上下 padding **16px**。标题字号 16px，字重 600，颜色 `#1D2129`。包含 14px 关闭按钮。
  - **内容区 (Body)**：左右 padding **20px**，上下 padding **24px**。字号 14px，文字 `#4E5969`。
  - **底部按钮区 (Footer)**：高度自适应，左右 padding **20px**，上下 padding **16px**。按钮靠右对齐。次要按钮在左，主确认按钮在右，横向间距 **8px**。
  - **【核心约束】**：头部、内容区、底部按钮区之间默认不设置可见的分隔线，但在内容长发生滚动时，内容区上下边缘出现 1px 的 `#F2F3F5` 分割线。

#### 4.5.4 Message & Notification 轻提示与通知
- **Message（全局轻提示）**：视口顶部居中，距离顶部 **16px**。高度固定为 **38px**，圆角 **4px**，内边距 `8px 16px`。白底背景，带轻投影，字号 14px。
- **Notification（通知提醒框）**：视口右上角拉出，距离顶部 **16px**，距离右侧 **16px**。宽度固定为 **320px**，圆角 **4px**，带标准阴影，内边距为 `16px 20px`。包含状态图标、标题（14px，`#1D2129`，加粗）和描述内容（13px，`#4E5969`）。

#### 4.5.5 Popconfirm 气泡确认框
- 悬浮于操作按钮上方的气泡式二次确认框。
- 宽度固定为 **200px**，白底圆角 **4px**，带轻投影。
- 内部上部展示警告图标与确认文案（13px，`#1D2129`），下部展示确定与取消按钮（小尺寸，高 24px），间距为 8px，内边距统一为 `12px 16px`。

## 6. Agent 极致自检清单 (Self-Checklist)

Agent 在编写、重构或校验任何 PC 后台页面的样式时，必须逐项核对本清单，并在输出结果中汇报自检情况：

- [ ] **颜色自检**：是否已全部将旧品牌色 `#3876FA` 替换为新主色 `#165DFF`？中性文本是否仅使用了 `#1D2129`、`#4E5969` 和 `#86909C`？边框与分割线是否分别为 `#E5E6EB` 和 `#F2F3F5`？
- [ ] **圆角自检**：输入框 (Input)、下拉框 (Select)、单选复选框、页码按钮、标准按钮的圆角是否全为 `4px`？卡片、Modal 容器、Drawer 容器的圆角是否全为 `8px`？
- [ ] **间距自检**：
  - 按钮与按钮横向间距是否固定为 `8px`？
  - 表单项（Form Item）上下间距：普通项是否为 `16px`？有校验报错项是否预留了 `24px` 垂直间距以防文案展示时页面抖动？
  - 卡片与卡片垂直间距是否固定为 `16px`？
  - 最外层页面容器 padding 是否固定为 `24px`？
- [ ] **字号自检**：页面主标题是否为 `18px`，卡片标题是否为 `16px`（左侧带 5px × 18px 竖条），常规正文字号是否为 `14px`？
- [ ] **表格对齐自检**：表格每一列，其“表头文字”与“表体内容”的对齐方式是否绝对一致？是否有“表头居中，表体左对齐”的错位现象？
- [ ] **表格宽度自检**：表格是否为 `100%` 自适应宽度？是否给序号、多选框、操作列、时间等核心列指定了 `min-width`，防止屏幕缩小时排版变形？
- [ ] **常用控件尺寸自检**：Input、Select、DatePicker 的高度是否均为 `32px`？Switch 的尺寸是否为 `22px × 40px`？
- [ ] **气泡与弹窗自检**：Modal 与 Drawer 遮罩层是否为 `#1D2129` 60% 透明度？Modal 底部按钮是否靠右，且间距为 `8px`？
